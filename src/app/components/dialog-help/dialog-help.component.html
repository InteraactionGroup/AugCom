Comment fonctionne cette application?

Lecture

En haut à droite vous avez le bouton play.

Lorsque vous cliquez dessus, il lit les mots dans l'ordre qui se trouve dans la barre.

Retour

En haut à droite vous avez le bouton retour.

Lorsque vous cliquez dessus, il efface le dernier mot que vous avez mis dans la barre.

Effacer

En haut à droite vous avez le bouton clear.

Lorsque vous cliquez dessus, il efface tout ce que vous avez mis dans la barre.

Retour dossier

En bas à gauche vous avez le bouton retour dossier.

Lorsque vous cliquez dessus vous revenez au dossier précédent.

Verrouiller/Déverrouiller

En bas à droite vous avez le bouton verrou.

Lorsque vous cliquez dessus vous verrouillez/déverrouillez les fonctionnalités ci-dessous.

Options

En bas à droite vous avez le bouton options.

Lorsque vous cliquez dessus vous allez dans les options.

Edition

En bas à droite vous avez le bouton d'édition.

Lorsque vous cliquez dessus vous pouvez ajouter, supprimer, modifier, déplacer les boutons.

Recherche

En bas à gauche vous avez le bouton de recherche.

Lorsque vous cliquez sur la flèche à sa droite vous pouvez rechercher un bouton présent dans la page ou les pages dans les dossiers.

Vue

En bas à droite vous avez le montrer/cacher.

Lorsque vous cliquez dessus cache les boutons qui ont le tag cacher, lorsque vous appuyez de nouveau, affiche tous les boutons de la grille

Plein écran

En bas à droite vous avez le bouton plein écran.

Lorsque vous cliquez dessus vous passer en plein écran, re-cliquez pour passer en mode fenêtré.

Langue

En bas à droite vous avez le bouton avec le drapeau de la langue utilisée.

Lorsque vous cliquez dessus vous passer vous échanger entre la langue courante avec l'anglais ou le français.
